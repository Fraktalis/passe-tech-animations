Le chiffrement expliqué

Modèle de miniature : dégradé rose→vert, titre court en Poppins ExtraBold, pictogramme 3D optionnel.

Guidelines rapides

  • Utilisez 8–12px d’espacement interne minimum dans les éléments, coins 18px.
  • Privilégiez des textes courts et contrastés. Évitez plus de 4 mots sur les miniatures.
  • Animations : transitions douces (200–300ms), lueurs légères sur les accents.
  • Compatibilité sombre/clair via la classe .theme-dark portée sur <body>.
